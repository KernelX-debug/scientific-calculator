# 🧮 Scientific Web Calculator

An **interactive scientific calculator** built with **HTML, CSS, and JavaScript**.  
It includes basic operations, scientific functions, and an **animated mathematical background**.

---

## 🚀 Features

### Basic operations
- Addition
- Subtraction
- Multiplication
- Division

### Scientific functions
- √ square root
- x² power
- sin
- cos
- tan
- log
- π

### Trigonometric mode
Trigonometric functions work in **degrees (sexagesimal mode)**.

Example:

| Operation | Result |
|-----------|--------|
| sin(30) | 0.5 |
| cos(60) | 0.5 |
| tan(45) | 1 |

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/scientific-calculator.git
```

Enter the project folder:

```bash
cd scientific-calculator
```

Open the file:

```
index.html
```

in your web browser.

---

## 💡 Usage

1. Enter numbers using the buttons.
2. Select the desired operation.
3. Press **=** to calculate the result.

Trigonometric functions operate in **degrees**.
